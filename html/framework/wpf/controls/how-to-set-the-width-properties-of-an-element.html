Практическое руководство. Определение свойств ширины элемента

Пример

В этом примере визуально показаны различия в поведении между четырьмя свойствами, связанными с шириной в отрисовки Windows Presentation Foundation (WPF).

FrameworkElement Класс предоставляет четыре свойства, описывающие характеристики ширины элемента. Эти четыре свойства могут конфликтовать и когда это значение, которое имеет более высокий приоритет определяется следующим образом: MinWidth значение имеет приоритет над MaxWidth значение, которое в свою очередь имеет приоритет над Width значение. Четвертое свойство ActualWidthдоступно только для чтения и сообщает фактическую ширину, определенной при взаимодействии с процессом макета.

Следующие Язык XAML примерах Rectangle элемент (rect1) как дочерний Canvas. Можно изменить свойства ширины Rectangle с помощью ряда ListBox элементы, представляющие значения свойств MinWidth, MaxWidth, и Width. Таким образом визуально отображается приоритет каждого свойства.

В следующих примерах кода обработки событий, SelectionChanged вызывает событие. Каждый пользовательский метод принимает входные данные из ListBox, анализирует его как Doubleи применяет значение к указанному свойству ширины. Значения ширины также преобразуются в строку и записываются в различные TextBlock элементы (определение этих элементов не представлено в выбранной XAML).

private void changeWidth(object sender, SelectionChangedEventArgs args)
{
    ListBoxItem li = ((sender as ListBox).SelectedItem as ListBoxItem);
    Double sz1 = Double.Parse(li.Content.ToString());
    rect1.Width = sz1;
    rect1.UpdateLayout();
    txt1.Text = "ActualWidth is set to " + rect1.ActualWidth;
    txt2.Text = "Width is set to " + rect1.Width;
    txt3.Text = "MinWidth is set to " + rect1.MinWidth;
    txt4.Text = "MaxWidth is set to " + rect1.MaxWidth;
}
private void changeMinWidth(object sender, SelectionChangedEventArgs args)
{
    ListBoxItem li = ((sender as ListBox).SelectedItem as ListBoxItem);
    Double sz1 = Double.Parse(li.Content.ToString());
    rect1.MinWidth = sz1;
    rect1.UpdateLayout();
    txt1.Text = "ActualWidth is set to " + rect1.ActualWidth;
    txt2.Text = "Width is set to " + rect1.Width;
    txt3.Text = "MinWidth is set to " + rect1.MinWidth;
    txt4.Text = "MaxWidth is set to " + rect1.MaxWidth;
}
private void changeMaxWidth(object sender, SelectionChangedEventArgs args)
{
    ListBoxItem li = ((sender as ListBox).SelectedItem as ListBoxItem);
    Double sz1 = Double.Parse(li.Content.ToString());
    rect1.MaxWidth = sz1;
    rect1.UpdateLayout();
    txt1.Text = "ActualWidth is set to " + rect1.ActualWidth;
    txt2.Text = "Width is set to " + rect1.Width;
    txt3.Text = "MinWidth is set to " + rect1.MinWidth;
    txt4.Text = "MaxWidth is set to " + rect1.MaxWidth;
}
