Практическое руководство. Получение или определение значения Dock

В следующем примере показано, как назначить Dock значение для объекта. В примере используется GetDock и SetDock методы DockPanel.

Пример

В примере создается экземпляр TextBlock элемент, txt1и назначает Dock значение Top с помощью SetDock метод DockPanel. Затем он устанавливает значение Dock свойства Text из TextBlock элемента с помощью GetDock метод. Наконец, в примере добавляется TextBlock элемент родителю DockPanel, dp1.

// Create the Panel DockPanel
dp1 = new DockPanel();

// Create a Text Control and then set its Dock property
txt1 = new TextBlock();
DockPanel.SetDock(txt1, System.Windows.Controls.Dock.Top);
txt1.Text = "The Dock Property is set to " + DockPanel.GetDock(txt1);
dp1.Children.Add(txt1);
mainWindow.Content = dp1;
mainWindow.Show();
