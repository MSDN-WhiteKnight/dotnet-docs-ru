Стили и шаблоны элемента Control

Элементы управления в Windows Presentation Foundation (WPF) имеют ControlTemplate , содержит визуальное дерево этого элемента управления. Можно изменить структуру и внешний вид элемента управления, изменив ControlTemplate этого элемента управления. Невозможно заменить только часть визуального дерева элемента управления; Чтобы изменить визуальное дерево элемента управления, необходимо задать Template свойство элемента управления, его нового и полного ControlTemplate.

Темы рабочего стола определяют, какой словарь ресурсов используется. Сведения о получении словарей ресурсов тем рабочего стола см. в разделе Темы WPF по умолчанию.
