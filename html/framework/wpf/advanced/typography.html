Оформление

Windows Presentation Foundation (WPF) включает поддержку форматированного представления текстового содержимого. Текст в WPF отрисовывается с помощью Технология Microsoft ClearType, что улучшает четкость и читаемость текста. WPF также поддерживает OpenType шрифты, которые обеспечивают дополнительные возможности помимо тех, которые определяются TrueType® формат.
