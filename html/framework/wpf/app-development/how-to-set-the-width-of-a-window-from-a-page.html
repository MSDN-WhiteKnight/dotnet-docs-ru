Практическое руководство. Задание ширины окна из страницы

В этом примере показано задание ширины окна из Page.

Пример

Объект Page можно задать ширину окна ее размещения, задав WindowWidth. Это свойство позволяет Page не были точно знать тип окна, на котором она размещена.

Note

Чтобы задать ширину окна с помощью WindowWidth, Page должен быть дочерним элементом окна.

<Page
    xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
    Title="SetWindowWidthPage"
    WindowWidth="500"
    >
</Page>
