Отображение справки по элементам управления с помощью всплывающих подсказок

Можно использовать ToolTip компонент для отображения кратких специализированных справочных сообщений для отдельных элементов управления в формах Windows Forms. ToolTip Компонент предоставляет свойство, указывающее текст, отображаемый для каждого элемента управления в форме.
