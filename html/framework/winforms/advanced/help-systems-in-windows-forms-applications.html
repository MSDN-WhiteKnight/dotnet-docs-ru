Справочные системы в приложениях Windows Forms

Один из наиболее важных преимуществ, как разработчик приложений, можно предоставить пользователям является исчерпывающая справочная система. Это, где они будет включить, когда они становятся путать или затруднений. Справочную систему в приложении Windows легко выполняется с помощью компонент HelpProvider.

Различные типы справки

Компонент HelpProvider Windows Forms позволяет связать HTML-файл справки версии 1.x (CHM-файл, созданный с помощью средства HTML Help Workshop, или HTM-файл) с приложением Windows. HelpProvider Компонент может использоваться для предоставления контекстной справки для элементов управления в формах Windows Forms или определенных элементов управления. Кроме того HelpProvider компонента можно открыть файл справки для конкретной области, например на главную страницу содержимого, индекса или для функции поиска. Общие сведения о HelpProvider компонента, см. в разделе Общие сведения о компоненте HelpProvider. Сведения о том, как использовать HelpProvider компонент для отображения всплывающей справки в формах Windows Forms, см. в разделе как: Отображение всплывающей справки. Дополнительные сведения об использовании ToolTip компонент для отображения справки для элемента управления, см. в разделе подсказки справки с помощью элемента управления.

Можно создать файлы HTML Help 1.x с HTML Help Workshop. Дополнительные сведения о HTML-справки см. в разделе «HTML Help Workshop» или в других разделах «HTML Help» в библиотеке MSDN.
