Приложения с интерфейсом MDI

Приложения многодокументного интерфейса (MDI) позволяют отображать несколько документов в то же время, при этом каждый документ отображается в отдельном окне. MDI приложения часто попадают в окно меню с вложенными меню для переключения между окнами или документами.

Note

Существуют некоторые поведения различия между MDI-форм и однооконным интерфейсом (SDI) windows в Windows Forms. Opacity Свойство не влияет на внешний вид дочерних MDI-форм. Кроме того CenterToParent метод не влияет на поведение дочерних MDI-форм.
