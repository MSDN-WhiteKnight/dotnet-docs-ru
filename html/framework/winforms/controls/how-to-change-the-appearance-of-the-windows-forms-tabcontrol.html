Практическое руководство. Изменение внешнего вида элемента управления TabControl в Windows Forms

Можно изменить внешний вид вкладок в формах Windows Forms с помощью свойств объекта TabControl и TabPage объекты, составляющие отдельные вкладки в элементе управления. Задавая эти свойства, можно отображать изображения на вкладках, отображение вкладок вертикально, отображения нескольких рядов и также включить или отключить вкладки программными средствами.

Для отображения значка со стороны метка вкладки

  1. Добавление ImageList на форму элемент управления.

  2. Добавление изображений в списке изображений.

    Дополнительные сведения о списках изображений, см. в разделе компонента ImageList и как: Добавление и удаление изображений с помощью Windows Forms компонента ImageList.

  3. Задайте ImageList свойство TabControl для ImageList элемента управления.

  4. Задайте ImageIndex свойство TabPage к индексу соответствующего изображения в списке.

Чтобы создать несколько строк вкладок

  1. Добавите число страниц вкладок, которые нужно.

  2. Задайте Multiline свойство TabControl для true.

  3. Если вкладки не отображается в нескольких строках, задайте Width свойство TabControl быть уже, чем все знаки табуляции.

Расположение вкладок вдоль края элемента управления

  • Задайте Alignment свойство TabControl для Left или Right.

Чтобы включить или отключить все элементы управления на вкладке

  1. Задайте Enabled свойство TabPage для true или false.

    TabPage1.Enabled = False  

    tabPage1.Enabled = false;  

    tabPage1->Enabled = false;

Отображение вкладок как кнопки

  • Задайте Appearance свойство TabControl для Buttons или FlatButtons.
