Элемент управления Panel (Windows Forms)

Windows Forms Panel элементы управления используются для предоставления возможность идентифицируемой группировки других элементов управления. Как правило панели используются для разделения формы функцией. Panel Управления аналогичен GroupBox управления; тем не менее, только Panel элемент управления может иметь полосы прокрутки и только GroupBox управления отображается заголовок.
