Практическое руководство. Определение размера панелей строки состояния

Note

Элемент управления ToolStripStatusLabel заменяет элемент управления StatusBar и расширяет его функциональные возможности; однако при необходимости элемент управления StatusBar можно сохранить для обратной совместимости и использования в будущем.

Каждый экземпляр StatusBarPanel класса в элемента управления StatusBar элемент управления имеет ряд динамических свойств, определяющих его ширину и изменение размеров во время выполнения.

Чтобы задать размер панели

  1. В процедуре, задайте AutoSize, MinWidth, и Width свойства (или любое подмножество этой последовательности) для строки состояния панели, с помощью индекса передать Panels свойство StatusBarPanel коллекции.

    Public Sub SetStatusBarPanelSize()  
' Create panel and set text property.  
   StatusBar1.Panels.Add("One")  
' Set properties of panels.  
   StatusBar1.Panels(0).AutoSize = StatusBarPanelAutoSize.Spring  
   StatusBar1.Panels(0).Width = 200  
' Enable the StatusBar control to display panels.  
   StatusBar1.ShowPanels = True  
    End Sub  

    public void SetStatusBarPanelSize()  
{  
   // Create panel and set text property.  
   statusBar1.Panels.Add("One");  
   // Set properties of panels.  
   statusBar1.Panels[0].AutoSize = StatusBarPanelAutoSize.Spring;  
   statusBar1.Panels[0].Width = 200;  
   statusBar1.ShowPanels = true;  
}  

    public:  
   void SetStatusBarPanelSize()  
   {  
      // Create panel and set text property.  
      statusBar1->Panels->Add("One");  
      // Set properties of panels.  
      statusBar1->Panels[0]->AutoSize =  
         StatusBarPanelAutoSize::Spring;  
      statusBar1->Panels[0]->Width = 200;  
      statusBar1->ShowPanels = true;  
   }
