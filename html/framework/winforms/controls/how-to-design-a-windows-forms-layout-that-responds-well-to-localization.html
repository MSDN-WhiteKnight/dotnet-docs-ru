Практическое руководство. Формирование макета формы Windows Forms с учетом будущей локализации

Создание готовых для локализации форм значительно ускоряет разработку продуктов для международных рынков. Элемент управления TableLayoutPanel можно использовать для реализации макетов, которые поддерживают изменение размера элементов управления при изменении значений их свойств Text.

Пример

Эта форма показывает, как создать макет, который пропорционально корректируется при отображении строковых значений на других языках. Такой процесс перевода называется локализацией. Подробнее об этом см. в разделе Локализация.

В Visual Studio предусмотрена расширенная поддержка данной задачи. См. также Пошаговое руководство: Создание структуры, сохраняющей пропорции при локализации.
