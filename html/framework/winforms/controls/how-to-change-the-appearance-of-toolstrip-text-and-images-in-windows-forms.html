Практическое руководство. Изменение внешнего вида текста и изображений элемента управления ToolStrip в Windows Forms

Можно управлять, отображаются ли текст и изображения на ToolStripItem и как они выравниваются относительно друг друга и ToolStrip.

Чтобы определить, отображаемые на ToolStripItem

  • Задайте DisplayStyle свойство нужное значение. Возможные значения — Image, ImageAndText, None, и Text. Значение по умолчанию — ImageAndText.

    ToolStripButton2.DisplayStyle = _  
    System.Windows.Forms.ToolStripItemDisplayStyle.Image  

    toolStripButton2.DisplayStyle = System.Windows.Forms.ToolStripItemDisplayStyle.Image;

Для выравнивания текста на ToolStripItem

  • Задайте TextAlign свойство нужное значение. Возможные значения — любое сочетание сверху "," Средний "и" нижний left, center и right. Значение по умолчанию — MiddleCenter.

    ToolStripSplitButton1.TextAlign = _  
    System.Drawing.ContentAlignment.MiddleRight  

    toolStripSplitButton1.TextAlign = System.Drawing.ContentAlignment.MiddleRight;

Чтобы выровнять изображение на ToolStripItem

  • Задайте ImageAlign свойство нужное значение. Возможные значения — любое сочетание сверху "," Средний "и" нижний left, center и right. Значение по умолчанию — MiddleLeft.

    ToolStripSplitButton1.ImageAlign = _  
    System.Drawing.ContentAlignment.MiddleRight  

    toolStripSplitButton1.ImageAlign = System.Drawing.ContentAlignment.MiddleRight;

Чтобы определить способ отображения выравниванием текста и изображения относительно друг друга

  • Задайте TextImageRelation свойство нужное значение. Возможные значения — ImageAboveText, ImageBeforeText, Overlay, TextAboveImage, и TextBeforeImage. Значение по умолчанию — ImageBeforeText.

    ToolStripButton1.TextImageRelation = _  
    System.Windows.Forms.TextImageRelation.ImageAboveText  

    toolStripButton1.TextImageRelation = System.Windows.Forms.TextImageRelation.ImageAboveText;
