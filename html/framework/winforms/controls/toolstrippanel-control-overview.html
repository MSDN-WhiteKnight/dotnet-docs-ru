Общие сведения об элементе управления ToolStripPanel

Объект ToolStripPanel предоставляет одну область для размещения и нависания ToolStrip, MenuStrip, и StatusStrip элементов управления. Несколько ToolStrip стека элементов управления по вертикали или по горизонтали в зависимости от Orientation из ToolStripPanel.

ToolStripPanel важные члены

name Описание
Orientation Возвращает или задает значение, указывающее горизонтальную или вертикальную ориентацию ToolStripPanel.
Renderer Возвращает или задает ToolStripRenderer используется для настройки внешнего вида ToolStripPanel.
RenderMode Возвращает или задает стили оформления элемента управления для применения к ToolStripPanel.
RowMargin Возвращает или задает расстояние в пикселях между ToolStripPanelRow и ToolStripPanel.
Rows Получает ToolStripPanelRow в этом ToolStripPanel.
Join Добавляет объект ToolStrip в ToolStripPanel.
