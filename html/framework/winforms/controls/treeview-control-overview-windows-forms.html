Общие сведения об элементе управления TreeView (Windows Forms)

С помощью элемента управления TreeView Windows Forms пользователям можно показывать иерархию узлов аналогично отображению файлов и папок на левой панели проводника операционной системы Windows. Каждый узел в представлении в виде дерева может содержать другие узлы, называемые дочерние узлы. Родительские узлы (т. е. содержащие дочерние узлы) можно показывать как в развернутом, так и в свернутом виде. В представлении в виде дерева рядом с узлами можно также показывать флажки, если присвоить свойству CheckBoxes этого дерева значение true. Вы можете выделять узлы и отменять их выделение программными средствами, задавая для свойства Checked узла значение true или false.

Ключевые свойства

Ключевые свойства элемента управления TreeView — это Nodes и SelectedNode. Свойство Nodes содержит список узлов верхнего уровня в представлении дерева. Свойство SelectedNode определяет выделенный в данный момент узел. Рядом с узлами могут отображаться значки. Элемент управления использует изображения из элемента ImageList, указанного в свойстве ImageList дерева. Свойство ImageIndex задает изображение по умолчанию для узлов в представлении дерева. Дополнительные сведения о выводе изображений см. в разделе как: Определение значков для Windows Forms TreeView-элемент управления. Если вы используете Visual Studio 2005, имеется доступ к большой библиотеке стандартных изображений, которые можно использовать с TreeView элемента управления.
