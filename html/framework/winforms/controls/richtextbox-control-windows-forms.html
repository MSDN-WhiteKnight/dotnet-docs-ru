Элемент управления RichTextBox (Windows Forms)

Windows Forms RichTextBox элемент управления используется для отображения, ввода и обработки текста с форматированием. RichTextBox Управления следит за недопущением TextBox делает элемент управления, но можно также отображать шрифты, цвета и ссылки; загрузить текст и встроенные изображения из файла; действия отмены и повтора операций; изменения и находить указанные символы. RichTextBox Управления обычно используется для работы с текстом и отображения функций, как для текстовых редакторов, таких как Microsoft Word. Как TextBox элемента управления, RichTextBox элемент управления может отображать полосы прокрутки; но в отличие от TextBox элемента управления, он по умолчанию отображаются горизонтальные и вертикальные полосы прокрутки и имеет дополнительные параметры полосы прокрутки.
