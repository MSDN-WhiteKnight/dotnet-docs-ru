Практическое руководство. Добавление и удаление вкладок с помощью элемента управления TabControl в Windows Forms

По умолчанию TabControl управления содержит два TabPage элементов управления. Получить доступ к этим вкладкам через TabPages свойство.

Чтобы добавить вкладку программными средствами

  • Используйте Add метод TabPages свойство.

    Dim myTabPage As New TabPage()  
myTabPage.Text = "TabPage" & (TabControl1.TabPages.Count + 1)  
TabControl1.TabPages.Add(myTabPage)  

    string title = "TabPage " + (tabControl1.TabCount + 1).ToString();  
TabPage myTabPage = new TabPage(title);  
tabControl1.TabPages.Add(myTabPage);  

    String^ title = String::Concat("TabPage ",  
   (tabControl1->TabCount + 1).ToString());  
TabPage^ myTabPage = gcnew TabPage(title);  
tabControl1->TabPages->Add(myTabPage);

Чтобы программно удалить вкладку

  • Чтобы удалить выбранных вкладок, используйте Remove метод TabPages свойство.

    -или-

  • Чтобы удалить все вкладки, используйте Clear метод TabPages свойство.

    ' Removes the selected tab:  
TabControl1.TabPages.Remove(TabControl1.SelectedTab)  
' Removes all the tabs:  
TabControl1.TabPages.Clear()  

    // Removes the selected tab:  
tabControl1.TabPages.Remove(tabControl1.SelectedTab);  
// Removes all the tabs:  
tabControl1.TabPages.Clear();  

    // Removes the selected tab:  
tabControl1->TabPages->Remove(tabControl1->SelectedTab);  
// Removes all the tabs:  
tabControl1->TabPages->Clear();
