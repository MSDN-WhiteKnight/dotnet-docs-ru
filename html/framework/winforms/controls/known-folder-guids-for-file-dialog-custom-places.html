Известные GUID папок для пользовательских размещений, отображаемых в диалоговом окне открытия или сохранения файла

Использовании Guid для указания Windows Vista известную папку при добавлении папок в CustomPlaces коллекции. Известные GUID папок не учитывается регистр и определены в файле KnownFolders.h в Windows SDK.

Note

В некоторых случаях известную папку добавляемого FileDialogCustomPlacesCollection не будет отображаться в Избранные ссылки области. Например если указанная известная папка отсутствует на компьютере, на котором выполняется приложение, известную папку не указывается.
