Как выполнить: Добавление пользовательского размещения в диалоговое окно сохранения или открытия файла

Диалоговые окна сохранения или открытия файла по умолчанию в Windows Vista содержат область в левой части окна под названием Избранные ссылки. Это область пользовательских размещений. OpenFileDialog И SaveFileDialog классы позволяют добавлять папки CustomPlaces коллекции.

Note

Чтобы пользовательское размещение отображалось в OpenFileDialog или SaveFileDialog, AutoUpgradeEnabled свойству должно быть присвоено true (по умолчанию).

Добавление пользовательского размещения в диалоговое окно сохранения или открытия файла

  • Добавьте путь, GUID известной папки, или FileDialogCustomPlace объект CustomPlaces коллекцию диалоговое окно.

    В следующем примере кода демонстрируется добавление пути.

    OpenFileDialog1.CustomPlaces.Add("C:\MyCustomPlace")  

    openFileDialog1.CustomPlaces.Add("C:\\MyCustomPlace");
