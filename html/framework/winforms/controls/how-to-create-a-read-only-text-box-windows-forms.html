Практическое руководство. Создать только для чтения текстовое поле (Windows Forms)

Изменяемое поле текста в Windows Forms можно преобразовать в элемент управления только для чтения. Например текстовое поле может отобразить значение обычно изменяется, но в настоящее время не может быть из-за состояния приложения.

Чтобы создать поле только для чтения текста

  1. Задайте TextBox элемента управления ReadOnly свойства true. Со свойством, имеющим значение true, пользователи могут прокручивать и выделите текст в текстовом поле, не позволяя изменять. Объект копирования команда работает в текстовом поле, но Вырезать и вставить команд не.

    Note

    ReadOnly Свойство оказывает влияние на взаимодействие с пользователем во время выполнения. По-прежнему изменением содержимого текстового поля программными средствами во время выполнения, изменив Text свойства текстового поля.
