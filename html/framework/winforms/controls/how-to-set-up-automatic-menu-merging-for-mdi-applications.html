Практическое руководство. Автоматическое слияние меню в приложениях MDI

В следующей процедуре представлены основные шаги по настройке автоматического слияния в приложении многодокументного интерфейса (MDI) с MenuStrip.

Чтобы задать автоматическое слияние меню

  1. Создание родительской MDI-формы, задав его IsMdiContainer свойства true.

  2. Добавить MenuStrip для родительской формы MDI, установка его MainMenuStrip свойства MenuStrip.

  3. Создание дочерних MDI-формы и задание его MdiParent имя родительской формы.

  4. Добавление MenuStrip для дочерней формы MDI.

  5. На дочерней формы, задайте Visible свойство MenuStrip для false.

  6. Добавление элементов меню в форме дочерних MenuStrip , которые необходимо объединить в родительской формы MenuStrip при активации дочерней формы.

  7. Используйте MergeAction свойства в меню элементов в форме дочерних MenuStrip для управления слиянием в родительской формы.
