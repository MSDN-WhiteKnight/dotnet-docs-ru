Практическое руководство. Добавление и удаление узлов элемента управления TreeView в Windows Forms

Windows Forms TreeView элемент управления сохраняет узлы верхнего уровня в его Nodes коллекции. Каждый TreeNode также имеет свой собственный Nodes коллекцию для хранения дочерних узлов. Оба свойства коллекции имеют тип TreeNodeCollection, который предоставляет стандартные элементы коллекции, которые позволяют добавлять, удалять и изменения порядка узлов на одном уровне иерархии узла.

Чтобы добавить узлы программным способом

  1. Используйте Add метод представления дерева Nodes свойство.

    ' Adds new node as a child node of the currently selected node.  
Dim newNode As TreeNode = New TreeNode("Text for new node")  
TreeView1.SelectedNode.Nodes.Add(newNode)  

    // Adds new node as a child node of the currently selected node.  
TreeNode newNode = new TreeNode("Text for new node");  
treeView1.SelectedNode.Nodes.Add(newNode);  

    // Adds new node as a child node of the currently selected node.  
TreeNode ^ newNode = new TreeNode("Text for new node");  
treeView1->SelectedNode->Nodes->Add(newNode);

Для удаления узлов программным способом

  1. Используйте Remove метод представления дерева Nodes свойство, чтобы удалить только один узел, или Clear метод, чтобы удалить все узлы.

    ' Removes currently selected node, or root if nothing is selected.  
TreeView1.Nodes.Remove(TreeView1.SelectedNode)  
' Clears all nodes.  
TreeView1.Nodes.Clear()  

    // Removes currently selected node, or root if nothing   
// is selected.  
treeView1.Nodes.Remove(treeView1.SelectedNode);  
// Clears all nodes.  
TreeView1.Nodes.Clear();  

    // Removes currently selected node, or root if nothing  
// is selected.  
treeView1->Nodes->Remove(treeView1->SelectedNode);  
// Clears all nodes.  
treeView1->Nodes->Clear();
