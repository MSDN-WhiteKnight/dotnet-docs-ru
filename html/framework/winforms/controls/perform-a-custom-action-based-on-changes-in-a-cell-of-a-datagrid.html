DataGridView Элемент управления имеет ряд событий, которые можно использовать для обнаружения изменений в состоянии DataGridView ячеек. Два из наиболее часто используемым CellValueChanged и CellStateChanged события.

Для обнаружения изменений значений ячеек элемента управления DataGridView

  • Создайте обработчик для CellValueChanged событий.

    private void dataGridView1_CellValueChanged(object sender,
    DataGridViewCellEventArgs e)
{
    string msg = String.Format(
        "Cell at row {0}, column {1} value changed",
        e.RowIndex, e.ColumnIndex);
    MessageBox.Show(msg, "Cell Value Changed");
}
