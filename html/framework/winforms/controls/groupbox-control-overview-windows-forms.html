Общие сведения об элементе управления GroupBox (Windows Forms)

Windows Forms GroupBox элементы управления используются для предоставления возможность идентифицируемой группировки других элементов управления. Как правило группы используются для разделения формы функцией. Например возможно с формой заказа, указаны параметры доставки почты ночное для использования. Объединение всех параметров в группу дает пользователю логическую визуальную подсказку, а во время разработки все элементы управления можно легко переместить — при перемещении одного GroupBox контролируете, все его вложенные элементы управления.

Заголовок «группы» определяется Text свойство. Дополнительные сведения см. в разделе Как Задать текст, отображаемый элементом управления форм Windows.

GroupBox и панели

GroupBox Управления аналогичен Panel управления; тем не менее, только GroupBox управления отображается заголовок и только Panel управления можно использовать полосы прокрутки.
