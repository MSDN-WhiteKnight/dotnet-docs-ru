Общие сведения о компоненте Timer (Windows Forms)

Компонент Windows Forms Timer вызывает событие через определенные интервалы времени. Этот компонент предназначен для работы в среде Windows Forms. Если вам требуется таймер, подходящий для серверной среды, см. раздел Общие сведения о серверных таймерах.

Ключевые свойства, методы и события

Длина в интервале, заданном параметром Interval свойство, значение которого указывается в миллисекундах. При включении компонента Tick события во время каждого интервала. Это, где необходимо добавить код, который будет выполнен. Дополнительные сведения см. в разделе Как Выполните процедуры, заданной периодичностью с помощью компонента Timer в Windows Forms. Основные методы класса Timer компонент Start и Stop, который включать таймер и отключать. При выключении таймера, он сбрасывает; Невозможно приостановить Timer компонента.
