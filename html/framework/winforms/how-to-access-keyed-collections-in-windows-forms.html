Практическое руководство. Доступ к коллекциям с ключом в Windows Forms

  • Можно получить доступ к ее отдельные элементы по ключу. Эта функциональность была добавлена для многих классов коллекций, которые обычно используются приложениями Windows Forms. Ниже перечислены некоторые из классов коллекций, имеющих доступны коллекции с ключом.

  • ListView.ListViewItemCollection

  • ListViewItem.ListViewSubItemCollection

  • Control.ControlCollection

  • TabControl.TabPageCollection

  • TreeNodeCollection

Ключ, связанный с элементом в коллекции, обычно является имя элемента. Следующие процедуры показывают, как использовать классы коллекций для выполнения распространенных задач.

Чтобы найти и передать фокус для вложенного элемента управления в коллекции элементов управления

  • Используйте Find и Focus методы, чтобы указать имя элемента управления, чтобы найти и передать фокус.

    OrderForm OrderForm1 = new OrderForm();
OrderForm1.Show();
OrderForm1.Controls.Find("textBox1", true)[0].Focus();

Для доступа к изображению в коллекции изображений

  • Используйте Item[] свойство, чтобы указать имя образа, необходимо получить доступ.

    this.BackgroundImage = imageList1.Images["logo.gif"];

Чтобы задать страницу вкладки в качестве выбранной вкладки

  • Используйте SelectedTab свойство вместе с Item[] свойство, чтобы указать имя страницы вкладки, чтобы задать в качестве выбранной вкладки.

    tabControl1.SelectedTab = tabControl1.TabPages["shippingOptions"];
