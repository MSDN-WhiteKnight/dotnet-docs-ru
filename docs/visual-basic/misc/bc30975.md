# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Управляющая переменная цикла For уже используется вложенным циклом For

Все вложенные циклы `For` должны использовать уникальные управляющие переменные.  
  
 **Идентификатор ошибки:** BC30975  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Измените имя одной из управляющих переменных.
