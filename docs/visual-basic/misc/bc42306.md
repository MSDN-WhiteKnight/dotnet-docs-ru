# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>Тег комментария XML "\<tag>" недопустим для элемента языка "\<element>"

Тег комментария XML \<tag> не разрешен для \<element> элемента "". Комментарий XML будет проигнорирован.  
  
 Тег комментариев XML не поддерживается элементом языка, к которому он применяется.  
  
 **Идентификатор ошибки:** BC42306  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите неподдерживаемый тег комментариев XML.
