# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Доступ к свойству должен выполняться для присваивания или считывания его значения

Предпринята попытка доступа к свойству без присваивания ему значения или использования его значения.
  
 **Идентификатор ошибки:** BC30545  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Присвойте значение свойству.  
  
     \- или -  
  
- Используйте значение свойства.
