# <a name="end-sub-must-be-the-first-statement-on-a-line"></a>"End Sub" должен являться первым оператором в строке

Оператор `End Sub` следует за разделителем операторов в виде двоеточия (:). `End Sub` должен быть единственным оператором в своей строке исходного кода.  
  
 **Идентификатор ошибки:** BC32031  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Разместите несколько операторов в отдельных строках.
