# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Невозможно прочитать поток, переданный в TextFieldParser.

[TextFieldParser Object](../language-reference/objects/textfieldparser-object.md) не может прочитать поток, который был в него передан. Это может быть результатом попытки чтения файла, который не является текстовым.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что файл является текстовым файлом.
