# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>Целевая версия .NET Compact Framework не поддерживает использование модификатора Ansi, Auto или Unicode

Версия .NET Compact Framework, с которой вы работаете, не поддерживает модификаторы `Ansi`, `Auto`и `Unicode` .  
  
 **Идентификатор ошибки:** BC30763  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите модификатор.
