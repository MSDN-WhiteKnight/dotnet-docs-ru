# <a name="trycast-operand-must-be-reference-type-but-typename-is-a-value-type"></a>Операнд "TryCast" должен быть ссылочного типа, а "\<typename>" является типом значения

Оператор `TryCast` используется с типом значения для по крайней мере одного из аргументов.  
  
 `TryCast` проверяет отношение наследования или реализации между двумя аргументами. Таким образом, он допускает только ссылочные типы аргументов. Дополнительные сведения см. в разделе [типы значений и ссылочные типы](../programming-guide/language-features/data-types/value-types-and-reference-types.md).  
  
 **Идентификатор ошибки:** BC30792  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Используйте оператор `DirectCast` или `CType` для выполнения преобразования. Они оба допускают типы значений.
