# <a name="file-filename-could-not-be-found"></a>Не удалось найти файл \<filename>

Не удалось найти указанный файл. Это может быть результатом указания неправильного пути к файлу.  
  
 **Идентификатор ошибки:** BC2001  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Проверьте правильность имени файла и пути.
