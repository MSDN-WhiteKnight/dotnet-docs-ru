# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Блок комментариев XML не может быть связан ни с одним элементом языка, который поддерживает применение комментариев документа XML.

Блок комментариев XML не удается связать с каким-либо допустимым элементом языка. XML-комментарий XML будет игнорироваться. Эта ошибка также происходит, если XML-тег в блоке комментариев не содержит соответствующий элемент языка.  
  
 **Идентификатор ошибки:** BC42312  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Найдите и удалите лишний XML-тег или предоставьте описываемый элемент языка.
