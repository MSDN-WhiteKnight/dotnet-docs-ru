# <a name="compiler-error-cs2001"></a>Ошибка компилятора CS2001

Не удалось найти исходный файл "файл"  
  
 Имя файла исходного кода было передано компилятору, но найти этот файл не удалось. Проверьте правильность написания имени файла и его расположение.
