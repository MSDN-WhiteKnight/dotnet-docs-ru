# \<policyImporters>

Этот элемент конфигурации задает все средства импорта политики, контролирующие импорт утверждений пользовательской политики о привязках. Каждый дочерний элемент — это <`policyImporter`>, который используется для поиска утверждений пользовательской политики о функциях привязки, а также для присоединения пользовательского элемента привязки, реализующего функции, необходимые для утверждения.
