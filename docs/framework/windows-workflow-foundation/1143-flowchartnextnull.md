# <a name="1143---flowchartnextnull"></a>1143 - FlowchartNextNull

## <a name="properties"></a>Свойства  
  
|||  
|-|-|  
|ID|1143|  
|Keywords|WFActivities|  
|Уровень|Сведения|  
|Канал|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Описание  

 Указывает, что следующий узел в блок-схеме NULL. Выполнение блок-схемы будет завершено.  
  
## <a name="message"></a>Сообщение  

 Блок-схема «%1»/FlowStep - следующий узел имеет значение NULL. Выполнение блок-схемы будет завершено.  
  
## <a name="details"></a>Сведения  
  
|Имя элемента данных|Тип элемента данных|Описание|  
|--------------------|--------------------|-----------------|  
|Блок-схема|xs:string|Отображаемое имя блок-схемы.|  
|Домен приложения|xs:string|Строка, возвращаемая AppDomain.CurrentDomain.FriendlyName.|
