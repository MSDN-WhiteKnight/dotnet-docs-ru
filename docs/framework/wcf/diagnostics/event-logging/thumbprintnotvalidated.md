# <a name="thumbprintnotvalidated"></a>ThumbPrintNotValidated

Идентификатор: 153  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## <a name="description"></a>Описание  

 Это событие указывает, что не удалось проверить сертификат удостоверения с заданным отпечатком. В событии указаны имя и ИД процесса.
