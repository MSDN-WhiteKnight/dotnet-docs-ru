# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed

Идентификатор: 127  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание  

 Это событие показывает, что регистрация поставщика WMI не отменена. В событии указаны объект WMI, ошибка, имя процесса и идентификатор процесса.
