# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose

System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Описание  

 Истекло время ожидания операции закрытия ServiceHost. Это означает, что некоторые каналы сеансов не были правильно завершены клиентом и могли быть прерваны. Измените время по умолчанию, чтобы отрегулировать время ожидания закрытия.
