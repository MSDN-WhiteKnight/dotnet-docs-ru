# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC

Когда был закрыт сеанс, была выполнена специальная транзакция.  
  
## <a name="description"></a>Описание  

 Трассируется, когда текущий активный сеанс был закрыт и для `TransactionAutoCompleteOnSessionClose` задано значение `true`.
