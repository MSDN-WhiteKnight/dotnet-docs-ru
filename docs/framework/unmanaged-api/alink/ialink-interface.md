# <a name="ialink-interface"></a>Интерфейс IALink

Помогает при построении сборок платформа .NET Framework. Помимо прочего, интерфейс содержит методы, которые помогают в написании манифестов сборок для многомодульных сборок, подписи сборок со строгими именами и создания модулей NETMODULE.
