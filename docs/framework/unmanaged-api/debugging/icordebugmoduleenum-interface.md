# <a name="icordebugmoduleenum-interface"></a>Интерфейс ICorDebugModuleEnum

Реализует методы ICorDebugEnum и перечисляет ICorDebugModule массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Следующий метод](icordebugmoduleenum-next-method.md)|Возвращает указанное количество `ICorDebugModule` экземпляров из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Remarks  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
