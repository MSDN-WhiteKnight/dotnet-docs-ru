# <a name="icorprofilerfunctionenumreset-method"></a>Метод ICorProfilerFunctionEnum::Reset

Перемещает курсор перечислителя в начальную позицию последовательности.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Reset();  
```
